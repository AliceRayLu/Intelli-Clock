# ZenLife 智能生活助手

一个全面的、支持语音交互的生活节律管理系统，旨在让您的一天更加和谐。ZenLife 助手将智能闹钟、睡眠辅助、冥想指导和番茄工作法集成于一体，利用 AI 技术提供个性化内容，带来无缝体验。

## ✨ 主要功能

### 1. 智能起床唤醒服务
- **智能闹铃**: 在设定时间用铃声唤醒您。
- **AI 新闻播报**: 关闭闹铃后，自动播放由大模型抓取并生成的语音新闻简报（天气、科技、政治、经济）。
- **个性化设置**:
  - **设置时间**: 通过语音修改时间；默认延续前一天设置。
  - **设置铃声**: 选择铃声强度（舒缓/强烈）；AI 根据强度自动寻找合适铃声。
- **关闭方式**:
  - 默认：通过按压/触摸关闭。
  - 支持语音命令"延迟几分钟"再次提醒。
  - 关闭闹钟后，自动开始播放新闻。

### 2. 睡眠提醒服务
- **轻柔提醒**: 在预设入睡时间前30分钟和到达入睡时间时，用舒缓的铃声进行提醒。
- **助眠音频**: 到达入睡时间时，自动播放由大模型生成的助眠音频，持续5分钟后停止。
- **个性化设置**:
  - **设置时间**: 通过语音设置入睡时间。
- **关闭方式**:
  - 默认：通过按压/触摸关闭提醒。
  - 支持语音命令"延迟几分钟"再次提醒。

### 3. 冥想服务
- **语音唤起**: 只需说出指令即可开始冥想。
- **灵活定时**: 通过语音或触摸设置冥想时长（默认10分钟）。
- **指导选项**: 可选择有指令引导的冥想或仅有背景音乐的无指令冥想。
- **温和结束**: 冥想结束时，以柔和的铃声唤醒。

### 4. 番茄钟服务
- **语音唤起**: 通过语音命令开始一个专注时段。
- **标准节奏**: 专注25分钟后休息5分钟；完成4个番茄钟后休息15分钟。
- **背景音效**: 在专注时段可选播放白噪音。
- **进度统计**: 查看今日专注总时长。
- **灵活操作**: 允许中途关闭番茄钟。

## 🚀 开始使用

### 环境要求
vscode 插件：ESP-IDF sdk v5.5.1

## 🗣️ 如何使用

主要的交互方式是**语音命令**。运行程序后，您可以说出如下指令：

*   **"把起床时间设为早上7点。"**
*   **"把我的闹钟铃声改成强烈的。"**
*   **"我想冥想15分钟，需要有指导。"**
*   **"开始一个番茄钟。"**
*   **"延迟5分钟提醒。"**
